# DigiTools - Premium Digital tools marketplace

A morderm digital digital tool marketplace build with react and talwind css where user 
can brwse, select, and perchess premium digital products.


## Technologies Used

1. **REact.js**
2. **Tailwind CSS**
3. **DaisyUI**
4. **Java Script ES6+**
5. **React Toastify**
6. **JSON**
7. **Vite**

## Key Features

1. **Product catalog with card Toggle**
2. **Real time Cart management**
3. **Toast Notifications**
